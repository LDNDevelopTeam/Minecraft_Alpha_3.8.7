## How to Run

### Client
**Requirements:** Java 8 (Zulu recommended)  
[Download Zulu Java 8](https://www.azul.com/downloads/?version=java-8-lts&os=windows&architecture=x86-64-bit&package=jdk#zulu)

1. Download the client zip file:  
[Alpha 3.8.7.zip](https://github.com/LDNDevelopTeam/Minecraft_Alpha_3.8.7/releases) (top file)

2. Install **MultiMC** or **PrismLauncher**:  
- [MultiMC](https://multimc.org/#Download)  
- [PrismLauncher](https://prismlauncher.org)

   After installation, log in with your Minecraft account and **drag & drop** the downloaded zip file into the launcher.

3. After installation, set the Java path to the Zulu Java 8 you installed. Adjust memory allocation as needed, then click Launch.

4. Enjoy playing!

---

Works like a normal Minecraft server. Please look up instructions yourself.  
Note: Server behavior differs from the client, and it is not perfect—expect bugs.
